CEASE

LTBEEF

LTBEEF (Literally The Best Exploit Ever Found) is a exploit found by bypassi (Bypassi#7037) in September 2022, and is a great way to disable spyware that was installed on your chromebook by your SysAdmins

Skiovox

Skiovox is an exploit that allows for browsing within a completely unblocked Chrome browser using a Kiosk App.

DATE

DATE (Dumb Alternative For an Exploit) is an exploit found by X-88 which gives you unblocked Google using College Board.

LTMEAT

LTMEAT (Literally The Meatiest Exploit of All Time) is an exploit discovered by Bypassi#7037 to bypass admin-forced extensions on managed Chromebooks.

Dextensify

Dextensify is an exploit which lets you disable most admin-installed Chrome extensions from any webpage.

Ingot

Ingot is a bookmarklet made by Fog Network to disable extensions based on LTBEEF with an interface based on the chrome extension page

Extension Installer

Extension Installer is a bookmarklet made by AkaButNice that lets you install extensions with the Chrome Web Store blocked.

INSECURLY

INSECURLY is an exploit made by Bypassi which allows you to toggle the Securly extension.

Pollen

Pollen is a system made by Mercury Workshop that can change Chrome User Policies enforced by your administrator.

LTMEAT (Flood)

This form of LTMEAT (Literally The Meatiest Exploit of All Time) is designed to activate the exploit without the use of debug URLs like LTMEAT does.

LTMEAT (Print)

This form of LTMEAT (Literally The Meatiest Exploit of All Time) is made to temporarily freeze extensions using the print dialog.
